The ASP API's `GET /{chainId}/public/mt-roots` returns two values: `mtRoot` (the ASP's latest computed root) and `onchainMtRoot` (the root currently committed on-chain). If `mtRoot !== onchainMtRoot`, the ASP has computed a new tree that has not been pushed on-chain yet — wait and re-fetch until they converge. Then confirm `BigInt(onchainMtRoot) === Entrypoint.latestRoot()` before generating a proof.
